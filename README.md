# MLB F5 Outcome Prediction Model

This repository contains a comprehensive end‑to‑end pipeline for building, training, and evaluating a probabilistic model to predict MLB **First Five Innings (F5)** outcomes.  The goal is to forecast whether the home or away team will lead after five innings and to estimate the likelihood of various F5 outcomes (home win, away win, tie) based on historical performance.

## Data

Historical MLB game logs and play‑by‑play data are pulled via the MLB Stats API and other open data sources.  The raw data is stored under `data/raw` and processed via scripts in `src/mlb` into the structured datasets found in `data/processed`.  Labels indicating F5 outcomes (home/away/tie) are generated by `src/mlb/build_f5_labels.py`.

## Features

The model uses a variety of engineered features to capture team strength and situational context, including:

- Offensive & defensive metrics (runs per game, OPS, wOBA)
- Pitching statistics (starter ERA, bullpen FIP, strikeout and walk rates)
- Park factors and weather
- Recent performance trends
- Rest days and travel distance

Feature generation functions are located in `src/features/`.

## Modeling Approach

The primary predictive model is a gradient boosted classifier trained to estimate the probability of each F5 outcome.  The pipeline supports experimentation with logistic regression, random forests, and gradient boosting (e.g., XGBoost/LightGBM).  Cross‑validation and hyperparameter tuning are implemented to guard against overfitting.  Evaluation metrics include accuracy, log‑loss, Brier score and calibration plots.

To reproduce the results:

1. Install dependencies: `pip install -r requirements.txt`
2. Generate labels and datasets: `python src/mlb/build_f5_labels.py && python src/mlb/build_model_dataset.py`
3. Train the model: `python src/models/train_f5_model.py`
4. Evaluate predictions: `python src/models/evaluate_f5_model.py`

## Results

In our baseline experiments, the gradient boosted model achieved ~56% accuracy on a hold‑out test set (random baseline = 33% in a three‑class problem) and demonstrated well‑calibrated probabilities across different seasons.  Detailed evaluation results and plots can be found in the `reports/` directory.

## How to Use

This project is intended for academic and research purposes.  To use the model for your own F5 outcome predictions:

- Update the raw data sources under `data/raw` with the most recent MLB season.
- Run the processing and model training scripts as described above.
- Use the trained model’s `predict_proba` method to obtain probability estimates for new games.
- Interpret the results within a proper research framework; they are **not** betting advice.
